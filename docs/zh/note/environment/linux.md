# Linux {#Linux}

> Linux 常用命令、服务器管理、开发环境配置等

## 常用命令 {#common-commands}

| 命令                             | 功能说明                                 |
| -------------------------------- | ---------------------------------------- |
| `mkdir include`                  | 创建一个 `include` 文件夹                |
| `unzip -d ./include include.zip` | 将 `include.zip` 解压到 `include` 文件夹 |
| `mv old_folder new_folder`       | 文件夹重命名                             |

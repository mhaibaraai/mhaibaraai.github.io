# Color

## getAnalogousColors

生成随机的类似色。

### 参数说明

| 参数             | 类型                                                                        | 默认值                                                    | 描述                   |
| ---------------- | --------------------------------------------------------------------------- | --------------------------------------------------------- | ---------------------- |
| **results**      | `number`                                                                    | `6`                                                       | 生成的颜色数量         |
| **slices**       | `number`                                                                    | `30`                                                      | 每个颜色生成的切片数量 |
| **colorSchemes** | `string[]`                                                                  | `['#8E44AD', '#F39C12', '#28B463', '#1F77B4', '#FF6F61']` | 使用的颜色方案         |
| **format**       | `'rgb' \| 'prgb' \| 'hex6' \| 'hex3' \| 'hex8' \| 'name' \| 'hsl' \| 'hsv'` | `'hex8'`                                                  | 输出颜色的格式         |

### 返回值

- **返回值类型**: `string[]`
  - 包含类似色的字符串数组，依据指定的 `format` 格式输出。

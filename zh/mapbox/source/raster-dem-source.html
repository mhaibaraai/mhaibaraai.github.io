RasterDemSource ​

RasterDemSource 一个栅格数字高程模型源。仅支持 Mapbox Terrain-DEM

在天地图上添加 3D 地形 ​

vue
<script lang="ts" setup>
import type { _MapboxProps } from '@movk-repo/mapbox'
import { Mapbox, RasterDemSource, Tianditu } from '@movk-repo/mapbox'

const accessToken = import.meta.env.VITE_MAPBOX_ACCESS_TOKEN
const token = import.meta.env.VITE_TIANDITU_TOKEN
const options = {
  zoom: 12.5,
  center: [117.12, 36.22],
  pitch: 77.28,
  bearing: -20,
} as _MapboxProps['options']
</script>

<template>
  <Mapbox :access-token="accessToken" :options="options">
    <Tianditu type="img_w" :token="token" />
    <RasterDemSource :exaggeration="1.5" />
  </Mapbox>
</template>

_RasterDemSourceProps ​

属性名说明类型默认值
id源的IDstringraster-dem-source_序号
options源的配置选项object-
url瓦片服务的URLstringmapbox://mapbox.mapbox-terrain-dem-v1
tiles瓦片源的数组array[]
tileSize瓦片大小（像素）number512
minzoom显示瓦片的最小缩放级别number0
maxzoom显示瓦片的最大缩放级别number14
exaggeration地形夸张高度number1
attribution瓦片源的名称string-
bounds四至边界范围array[-180, -85.051129, 180, 85.051129]
encoding源的数据编码enummapbox
volatile是否缓存瓦片booleanfalse
